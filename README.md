# Installation
## Requirements

Required:
* Android System Webview and/or Chrome
* Android version 5 or newer

# Building
## Requirements

Required
* Android Studio version 4.1.3
* Gradle Build Tools version 4.1.3
* Gradle version 6.8.1

# Downloading
git clone https://github.com/mrepol742/PROJECT-WEBVIUM
