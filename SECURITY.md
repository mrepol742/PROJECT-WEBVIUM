# Security Policy

## Supported Versions

* Security Vulnirability are been delivered via the latest update available and not by patch on each version.

## Reporting a Vulnerability

Please report (suspected) security vulnerabilities to
**[mrepol742@gmail.com](mailto:mrepol742@gmail.com)**. You will receive a response from
us as soon as possible. If the issue is confirmed, we will release the update as soon
as possible depending on complexity but historically within a few days.
